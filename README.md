# ToolUtils

## Some commonly used tool with android.

# utils/ #

简单一个文件即可实现对应需求。

## LogUtils.java

通过修改一个字段即可实现日志开关，方便上线前日志关闭。

## MD5Utils.java

获取字符串 MD5 文件。

## SPUtils.java

实现通过 SP 进行数据存取。

## ThreadUtils.java

通过线程池的方式创建新线程，减少每次 new 新线程的资源开销。

## ToastUtils.java

封装一个 Toast 类，方便子线程直接调用。

## EditTextClear.java

一个简单的自带删除按钮的文本输入框。

## SingletonTimer.java

全局倒计时统计单例类。

## ImageUtils.java

图片
